# Introduction

This repository includes the source to build a Docker image for an Ubuntu 18.04-based deployment of
ntpd, the Reference Implementation of the NTP protocol, by the Network Time Foundation.

# Prerequisites

* Docker installed

# Build

    ./scripts/build

# Run

    ./scripts/run

# Debug

    ./scripts/debug
